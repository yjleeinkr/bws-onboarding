# bws-onboarding

## 기본 동작

1. 신규 TodoList 추가
2. 추가된 TodoList의 완료 - 미완료 처리
3. 추가된 TodoList의 삭제

## 필수 사항

- ES6 문법을 사용해서 개발
- ES Module을 사용해서 컴포넌트 형태로 구현
- Web Storage를 사용해 TodoList 데이터 관리

## 선택 사항

- 드래그&드랍으로 Todo List 순서 변경 기능 추가
- 필터링 기능 ( 전부, 완료, 미완료 )
- 인라인 수정기능 ( 추가된 TodoList 더블클릭시 수정 가능하도록 )
